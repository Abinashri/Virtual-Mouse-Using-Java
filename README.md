# Virtual Mouse using Java

A computer vision-based project that allows users to control the mouse using hand gestures.

## 🚀 Features
- Control cursor movement using hand gestures
- Perform mouse clicks without physical mouse
- Real-time gesture tracking
- Improves accessibility and touchless interaction

## 🛠️ Tech Stack
- Java
- OpenCV (for image processing)
- Computer Vision concepts

## 📌 How It Works
- Webcam captures hand movements
- Image processing detects finger positions
- Hand gestures are mapped to mouse actions

## 📌 Example
- Move hand → Move cursor
- Tap gesture → Click
- Two fingers → Right click (optional feature)

## 🔮 Future Improvements
- Add gesture-based scrolling
- Improve accuracy using AI/ML
- Support more gestures
